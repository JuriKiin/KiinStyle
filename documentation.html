Color.

Below are the default text colors.

  • Grass Color
  • Water Color
  • Rock Color
  • Fire Color
  • Silk Color
  • Snow (White)
  • Coal Color

To set text color using these primary colors:

class='COLOR'

class='grass'

To set background color using these primary colors:

class='bg-COLOR'

class='bg-grass'

Width.

Width is used to set the width of an element.

  • auto
  • 10%
  • 20%
  • 25%
  • 50%
  • 75%
  • 100%

To set the width of an element:

class='item-NUMBER'

class='item-50'

Height.

Height controls the vertical height of an element.

Height exists in the following heights: 0px, 5px, 10px, 20px, 25px, 50px, 75px, 100px, 150px, auto

To set the height of an element:

class='height-NUMBER'

class='height-50'

Alignment.

Alignment is used to align an item left, right, or center.

  • Center
  • Left
  • Right

To set the alignment of an element:

class='align-DIRECTION'

class='align-center'

Margin.

Margins are used to create space around an element.

  • All
  • Left
  • Right
  • Top
  • Bottom

Margins exist in all directions in the following formats: 0, 10px (10p), 5%, 10%, 20%, 25%, 50%, 75%, 100%

To set the margins of an element:

class='margin-DIRECTION-NUMBER'

class='margin-top-50'

Padding.

Padding is used to generate space around an element.

  • All
  • Left
  • Right
  • Top
  • Bottom

Padding exists in all directions in the following formats: 0, 10px (10p), 20px (20p), 5%, 10%, 20%, 25%, 50%, 75%, 100%

To set the padding of an element:

class='pad-DIRECTION-NUMBER'

class='pad-top-50'

Font size controls the size of an element.

  • sx

  • lg
  • xl
  • xxl

To set the font size of an element:

class='font-SIZE'

class='font-lg'

Font Type.

Font size controls the size of an element.

  • Header
  • Text
  • Default

To set the font type of an element:

class='font-NAME'

class='font-header'

Line Height.

Line Height controls the height of a text element.

  • 0px
  • 5px
  • 10px
  • 20px
  • 25px
  • 50px
  • 75px
  • 100px
  • 150px

Line Height exists in the following sizes: 0px, 5px, 10px, 20px, 25px, 50px, 75px, 100px, 150px

To set the line height of an element:

class='l-height-NUMBER'

class='l-height-50'

Display.

Display effects the rendering box of the element.

  • Inline
  • Block
  • Inline-Block

Display also offers d-none, which sets the display of an element to none.

To set the display of an element:

class='d-NAME'

class='d-inline-block'

Float.

Float controls which way an element should float.

  • Left
  • Right

To set the float of an element:

class='f-DIRECTION'

class='f-left'

Links.

Links let the user move to a different page, or place on a page.

  • link (standard)
  • link-clear
  • link-hover
  • link-hover-subtle

Links have several default styles: link, link-clear, link-hover, link-hover-subtle

To style a link:

class='link-TYPE'

class='link-hover'
