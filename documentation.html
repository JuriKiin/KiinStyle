Backgrounds.

Background Images add popping content to a web page.

* You must add the background-image URL using inline HTML styles. You must also include a width and height. This can be done as such:

style="background-image: url('media/background.jpg'); width:100%; height:200px"

Background Image styles exist in the following: par, default

To add a background image:

class='bg-name'

class='bg-par'

Height.

Height controls the vertical height of an element.

Height exists in the following heights: 0px, 5px, 10px, 20px, 25px, 50px, 75px, 100px, 150px, auto

Height also exists in larger heights: full (100vh), half (50vh), quart (25vh), triquart (75vh)
