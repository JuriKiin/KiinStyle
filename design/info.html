Guide to Responsive Design.

Creating responsive design with KiinStyle is easy. Every class in the library exists with a mobile counterpart.

This means that if your design requires different styling on smaller screens, you can customize its appearance.

This is done by prefixing any style with m-.

Take the below example. The following code creates the below element:

<p class="font-lg pad-10p bg-sand snow height-75">This is a test paragraph.</p>

This is a test paragraph.

On a desktop, this might look okay, however on a mobile device, the height of this element might take up too much space.

In order to fix this, we can make use of the m- prefix, and set a new height for smaller screens:

Taking the above example, let's lower the height to something more reasonable by using m-height-20.

The full HTML declaration now looks like this:

<p class="font-lg pad-10p bg-sand snow height-75 m-height-20">This is a test paragraph.</p>

You can see that we keep the same height-75 that we want for wider screens, but now add the additional class m-height-20.

If you now look at the below UI element on a smaller screen, you'll notice that it takes on the smaller height, making it better.

This is a test paragraph.

If you are looking at this page on a computer screen, the two elements likely look the same. This is because the m- prefix only applies on smaller screens.

It's important to remember that the m- prefix can be applied to any style that you wish to change for smaller screens.
